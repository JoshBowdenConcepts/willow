# Willow Design System

Willow DS is a comprehensive design system built to provide a consistent, scalable foundation for digital products. It includes a robust set of design tokens for managing colors, typography, spacing, and elevation, ensuring visual coherence across all experiences. A curated icon library offers a flexible and accessible set of symbols to match a variety of use cases. At its core, Willow DS delivers a collection of reusable, accessible components, designed to be adaptable and easy to integrate into any product. Together, these elements empower teams to move faster, build with confidence, and create beautiful, cohesive user interfaces.

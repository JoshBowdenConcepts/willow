# Colors

Color tokens are the foundation of Willow DS’s color system. They represent the raw values for colors — such as brand colors, neutrals, and accent tones — abstracted into semantic, reusable variables. By using tokens instead of hard-coded color values, we ensure consistency, flexibility, and easier theming across different products and platforms.

Each token is named intentionally to reflect its role (for example, --color-background-primary or --color-text-muted), allowing teams to make design changes at scale without manually updating individual components. Tokens can be mapped to different color values based on context, such as light mode, dark mode, or brand-specific themes.

Organized into categories like Backgrounds, Text, Borders, and Interactive States, color tokens help create a cohesive, accessible visual language that adapts to user needs while maintaining a unified brand identity.
